# Ekster
Homepage and 3 advertorials pages for Ekster wallets.
Projects based on Nuxt.js
###### Demo: 
```
/ekster
/ekster/v1
/ekster/v2
/ekster/v3
```
## Dependencies
Use the package manager [npm](https://www.npmjs.com/) to install all dependencies.
```
npm install
```
## Start server and livereload
```
npm run dev
```
